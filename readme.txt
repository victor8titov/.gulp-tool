

Статус:		В процессе разработки.

Описание:	Конфигурационный файл для работы над проектами фронтенда. Настраивается под работу 
		с темой из разного уровня вложенности а так же для работы над одностраничными 
		сайтами.


Ошибки:		* сервер не синхронизирет файлы если папка разработки или буличная находится с наружи отностите
		  льно папки gulp

		* поменять имена фременных файлов на поддчеркивание в низу.

		* проверка папок gulp 
	          нет возможности работать с файлами less отдельно от general.less выдает ошибку при его отсутствии
